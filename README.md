# Real Estate Price Analysis (Екатеринбург)
Создание модели линейной регрессии для прогнозирования цены за квадратный метр квартиры и определения наиболее значимых факторов. В ноутбуке real_estate_analytics предстален ход анализа, фаил data_with_cords содержит в себе информацию с координатами обьекта, и, наконец, фаил data хранит основную информацию об обьектах. 

## Описание проекта

Проведён анализ рынка жилой недвижимости Екатеринбурга и разработана модель линейной регрессии для прогнозирования стоимости квадратного метра на основе характеристик квартиры и локации.

## Цель

Выявить ключевые факторы, влияющие на цену за м², и построить интерпретируемую модель для оценки стоимости объектов.

## Данные

Проанализировано ~1800 объявлений о продаже квартир (Циан):

* характеристики квартиры (площадь, комнаты, этаж, тип дома);
* год постройки;
* расстояния до центра, метро и аэропорта;
* географические координаты объектов.

## Используемый стек

Python

## Выполненные шаги

* Очистка данных: обработка пропусков, выбросов, корректировка типов.
* Feature engineering: подготовка признаков для моделирования.
* Исследовательский анализ рынка недвижимости.
* Оценка влияния локации и характеристик квартиры на цену.
* Построение модели линейной регрессии.
* Интерпретация коэффициентов модели для выявления драйверов стоимости.

## Ключевые результаты анализа

* Новостройки существенно дороже старого фонда.
* Расстояние до центра — один из сильнейших факторов снижения цены.
* Удалённость от метро влияет слабее, чем ожидалось.
* Большая площадь снижает цену за м² (эффект масштаба).
* Квартиры с большим числом комнат дешевле за м², чем студии и 1-комнатные.
* Высокие этажи и современные ЖК чаще связаны с более высокой стоимостью.
* Расстояние до аэропорта практически не влияет на цену.

## Итог

Модель линейной регрессии позволила количественно объяснить факторы ценообразования и может использоваться как базовый инструмент оценки стоимости недвижимости и анализа рынка.
